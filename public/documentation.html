MyFlix

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Businnes Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET none A JSON object holding data about all movies
Get data about a single movie by title /movies/[title] GET none A JSON object holding data about a single movie, containing description, genre, director, image URL, whether it’s featured or not. Example: {"title": "Split", "description": "The film follows a man with dissociative identity disorder who kidnaps...", "genre": { "name": "thriller", "description": "This film is a ..." }, "director": { "name": "M. Night Shyamalan", "bio": "Shyamalan was born in Mahé, India...", "birth": 1970 }, "imageURL": "https://www.google.com...", "featured": false}
Get data about genre /movies/genre/[genreName] GET none A JSON object holding data about a genre of each movie. Example: "genre": { "name": "drama", "description": "On top of all of that, the movie is a..." }
Get data about director /movies/director/[directorName] GET none A JSON object holding data about a director of each movie. Example: "director": { "name": "Luc Besson", "bio": "Besson was born in Paris...", "birth": 1959" }
Create a new user /users POST A JSON object holding data about a user to add, structured like: { "name": "Rita", "favoriteMovies": [] } A JSON object holding data about a user that was added, inluding an ID: { "name": "Rita", "favoriteMovies": [], "id": "54629442-cdea-4dbe-8f5a-2a3fa3fc9325" }
Remove a user by ID /users/[id] DELETE none A text message indicating whether a user was successfully removed.
Update a user's info (name) /users/[id] PUT A JSON object holding user's data that must be updated: { "name": "Iryna" } A JSON object holding data about a user that was updated: { "id": 1, "name": "Iryna", "favoriteMovies": [ "Leon" ] }
Add a movie to user favorite list /users/[id]/[movieTitle] PUT none A text message indicating whether a movie was successfully added.
Remove a movie from user favorite list /users/[id]/[movieTitle] DELETE none A text message indicating whether a movie was successfully removed.
